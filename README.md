# 淘宝购物车价格追踪器

自动抓取淘宝购物车商品价格，记录价格趋势，低价时提醒。

## 功能

- 🔄 自动抓取淘宝购物车的所有商品价格
- 📈 价格趋势图表：7天 / 30天 / 90天 / 180天 / 全部
- 📉 降价提醒：商品降价或达到历史最低价时通知
- 🔍 按商品名或店铺名实时搜索
- 📦 按店铺分组展示，与淘宝购物车一致
- ⚠️ 自动识别缺货/下架商品

## 准备条件

- **Windows 10/11** 电脑（系统自带 Microsoft Edge 浏览器，无需额外安装）
- 一个 **淘宝账号**

## 安装教程（一步一步来）

### 1. 安装 Python

打开浏览器访问：**https://www.python.org/downloads/**

点击黄色按钮下载，下载完成后双击安装。

⚠️ 安装时**一定要勾选底部 "Add Python to PATH"**，然后点 Install Now。

验证安装：按 `Win + R`，输入 `cmd` 回车，输入：

```bash
python --version
```

显示 `Python 3.x.x` 就对了。

### 2. 下载本项目

打开命令行（`Win + R`，输入 `cmd` 回车），输入：

```bash
git clone https://github.com/Aph-Potato/taobao-cart-tracker.git
cd taobao-cart-tracker
```

如果提示 `git 不是内部命令`，先装 Git：https://git-scm.com/download/win（一路 Next 即可）。

不会用 Git 也可以直接下载：点 GitHub 页面上的绿色 **Code** 按钮 → **Download ZIP** → 解压到任意文件夹。

### 3. 安装依赖

在项目文件夹里打开命令行，输入：

```bash
pip install -r requirements.txt
```

等待安装完成（约 1 分钟）。

### 4. 启动网站

```bash
python run.py
```

看到 `访问 http://localhost:8000` 就说明启动成功了。

打开浏览器，访问 **http://localhost:8000**，你会看到空白的仪表盘。

### 5. 首次抓取

点击页面上的 **🔄 立即抓取** 按钮。

会弹出一个 Edge 浏览器窗口，**用你的手机淘宝 App 扫屏幕上的二维码登录**。

登录成功后不要操作那个窗口，让它自动完成。几十秒后页面会提示"成功抓取 XX 个商品"。

刷新浏览器，商品就全部出现了。

## 日常使用

| 你想做什么 | 怎么做 |
|-----------|--------|
| 启动网站 | 双击项目里的 `start.bat`（桌面快捷方式也可以） |
| 打开网站 | 浏览器访问 `http://localhost:8000` |
| 查看价格 | 在仪表盘上直接看，点击商品名看趋势图 |
| 更新价格 | 点 **🔄 立即抓取** |
| 自动抓取 | 网站开着，每 N 小时自动抓一次（设置里改间隔） |
| 搜索商品 | 用页面顶部的搜索框，支持商品名和店铺名 |
| 降价提醒 | 设置里配置降幅阈值（0 = 只历史最低价时提醒） |
| 关掉网站 | 在启动的黑窗口里按 `Ctrl + C` |

## 注意事项

- 网站**必须一直运行**才能自动抓取（黑窗口别关，最小化就行）
- 抓取时不要操作弹出的 Edge 窗口，让它自动完成
- 如果弹出了淘宝滑块验证，手动拖一下就好
- 登录状态通常可以保持几天，过期后需要重新扫码
- **不要在短时间内频繁抓取**（比如几分钟一次），淘宝会认为是机器人
- 抓取间隔建议设置 6-24 小时

## 常见问题

**Q: 启动报错 "playwright not found"？**

重新执行 `pip install playwright`。

**Q: 抓取时 Edge 打开了但没反应？**

可能登录过期了，手动在淘宝扫码，然后关掉 Edge 窗口，再点一次立即抓取。

**Q: 商品图片加载不出来？**

正常现象，淘宝图片链接有时效限制，不影响价格追踪功能。

**Q: 想换一台电脑用，数据能迁移吗？**

复制项目文件夹里 `data/tracker.db` 文件到新电脑的同位置即可。
